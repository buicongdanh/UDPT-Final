## Getting Started
### Configuration
Cấu hình file config.inc.php user và mật khẩu của bạn

```PHP
<?php 
$config['database'] = [
    'host'      => 'localhost',
    'user'      => 'YOUR_USER_NAME',
    'password'  => 'YOUR_USER_PASSWORD',
    'db'        => 'albumdb'
];

```
